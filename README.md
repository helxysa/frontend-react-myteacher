<h1>Como iniciar projeto</h1>
<h2> Backend </h2>
<h3> Dependencias </h3>

```bash
 pip install
```
<p> Após isso instalar virtual venv </p>

```bash
 pip install virtualenv
```

<h2> Frontend </h2>
<h3> Dependencias </h3>

```bash
 npm install
```

<p> Após isso executar seguinte comando </p>

```bash
 npm run dev
```
